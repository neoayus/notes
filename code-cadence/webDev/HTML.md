# Boiler Plate Code 
`
```
<!DOCTYPE html>
<html>
    <head>
        <!-- head tag is used to store the metadata of the webpage. -->
        <title>title of this webpage </title>
    </head>
    <body>
        <p> This is a Boiler Plate code for html documents.  </p>
    </body>
</html>`
```
